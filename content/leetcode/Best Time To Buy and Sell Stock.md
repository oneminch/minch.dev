## My Solution

### JavaScript

```js
function maxProfit(prices: number[]): number {
  let min = prices[0],
    max = 0;

  for (let price of prices) {
    if (price < min) {
      min = price;
    }
    if (price - min > max) {
      max = price - min;
    }
  }

  return max;
}
```
